# Advanced HTML & CSS Project — README

## Overview

This project continues from the **HTML Advanced** tasks. Your goal is to style your page using CSS while following the design found in the provided Figma file.

## Objectives

- Recreate the layout from Figma as closely as possible.
- Use modern CSS (Flexbox, box model, custom fonts, colors, spacing).
- Keep HTML clean and semantic.
- Match fonts, sizes, spacing, and colors from the design.

## Figma

- Open the provided Figma link.
- Click **Duplicate to Drafts** to inspect spacing, colors, and typography.
- Download and install required fonts:

  - Source Sans Pro
  - Spin Cycle OT

## Assets

Download all project images and place them in an `images/` folder.
